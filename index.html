Qu'est-ce que l'Intelligence Artificielle?

Définition fondatrice : "La construction de programmes informatiques qui s'adonnent à des tâches qui sont, pour l'instant, accomplies de façon plus satisfaisante par des êtres humains car elles demandent des processus mentaux de haut niveau tels que : l'apprentissage perceptuel, l'organisation de la mémoire et le raisonnement critique."[1]

Définition contemporaine : "Ensemble de théories et de techniques mises en œuvre en vue de réaliser des machines capables de simuler l'intelligence humaine."[2]

Postulat de John McCarthy (Dartmouth, 1956): "Toute activité intellectuelle peut être décrite avec suffisamment de précision pour être simulée par une machine."[3]

Définitions différentes selon l'époque (contexte) et le type d'IA (forte ou faible).

Autres Approches en Intelligence Artificielle

IA Symbolique

  • Problèmes Traités: Raisonnement logique, diagnostic, planification.
  • Algorithme: Basée sur des règles logiques prédéfinies et la manipulation de symboles.
  • Exemples: Deep Blue, Prover9, Mycin.
  • Domaines d'Application: Diagnostic médical, aide à la décision.

IA Combinatoire

  • Problèmes Traités: Optimisation, planification, CSP.
  • Algorithme: Basée sur la recherche de solutions parmi un grand ensemble de combinaisons.
  • Exemples: Livraison Amazon, Google Map, Orange (réseaux).
  • Domaines d'Application: Transport, Aérospatiale, Social, Finance.

Machine Learning

Apprentissage automatique: Génère un modèle de prédiction à partir de patterns identifiés dans des données.

  • Types d'apprentissage en ML :
    • Apprentissage supervisé : le modèle est entraîné à partir de données étiquetées (ex. : classification).
    • Apprentissage non supervisé : le modèle apprend à partir de données non étiquetées (ex. : clustering).
    • Apprentissage par renforcement : l'algorithme apprend par essais et erreurs en recevant des récompenses.
  • Exemples d'algorithmes populaires :
    • Régression linéaire, arbres de décision, SVM, réseaux de neurones, etc.
  • Domaines d'application :
    • Reconnaissance d'image, traitement du langage naturel, systèmes de recommandation, etc.

Machine Learning

Apprentissage supervisé

  • Type de données : Étiquetées.
  • Le modèle apprend à partir de données avec des labels prédéfinis.
  • Exemples d'algorithmes : Régression linéaire, arbres de décision, support vector machines (SVM).
  • Applications : Reconnaissance d'images, prédiction de ventes, diagnostics médicaux.

Détection de spams

Apprentissage non supervisé

  • Type de données : Non étiquetées.
  • Le modèle cherche à trouver des structures cachées sans labels.
  • Exemples d'algorithmes : K-means, clustering hiérarchique, réseaux de neurones auto-encodeurs.
  • Applications : Segmentation de marché, regroupement de clients, analyse de documents.

Détection de spams

Apprentissage par renforcement

  • Type de données : Basé sur des récompenses et pénalités.
  • Le modèle interagit avec un environnement et apprend par essais et erreurs.
  • Exemples d'algorithmes : Q-learning, Deep Q-Network (DQN), SARSA.
  • Applications : Jeux vidéo, robots autonomes, voitures autonomes, trading automatisé.

Deep Learning

Le Deep Learning est une branche du Machine Learning qui utilise des réseaux de neurones artificiels profonds. Il se distingue par sa capacité à traiter de grandes quantités de données, en particulier pour des tâches complexes comme la reconnaissance d'image, le traitement du langage naturel et la traduction automatique.

  • Caractéristiques du Deep Learning :
    • Réseaux de neurones artificiels à plusieurs couches (réseaux neuronaux profonds).
    • Utilisation de grandes quantités de données et de calculs intensifs (GPU).
    • Algorithmes : backpropagation, convolutional neural networks (CNN), recurrent neural networks (RNN).
  • Applications courantes :
    • Reconnaissance vocale (ex. : assistants vocaux).
    • Traitement d'image et de vidéo (ex. : véhicules autonomes).
    • Traduction automatique (ex. : Google Translate).

Data Science

La Data Science est un domaine interdisciplinaire qui combine des techniques scientifiques, mathématiques et informatiques pour analyser des données complexes. Elle permet de découvrir des informations et des patterns dans les données pour soutenir la prise de décision.

  • Étapes du processus de Data Science :
    • Collecte des données.
    • Nettoyage et transformation des données.
    • Analyse exploratoire des données (EDA).
    • Modélisation statistique et machine learning.
    • Visualisation des résultats.
  • Outils populaires :
    • Python (Pandas, NumPy), R, SQL, Hadoop, Spark, etc.
  • Domaines d'application :
    • Marketing, santé, finance, industrie, etc.

Big Data

Le Big Data fait référence à l'ensemble des données massives et complexes qui ne peuvent pas être traitées par des méthodes traditionnelles. Il nécessite des technologies avancées pour la collecte, le stockage, et l'analyse de données en temps réel ou en différé.

  • Les 3 V du Big Data :
    • Volume : quantité énorme de données.
    • Vélocité : rapidité de génération des données.
    • Variété : diversité des types de données (structurées, non structurées).
  • Technologies associées :
    • Hadoop, Spark, NoSQL, bases de données distribuées.
  • Domaines d'application :
    • Analyse de sentiments, Internet des Objets (IoT), détection de fraudes, prédiction des tendances du marché.
